CBCT geometries and phantom
===========================

In the example, we have built a CBCT system with the following geometric parameters

.. list-table:: CBCT geometry
   :widths: 25 25
   :header-rows: 1

   * - Parameters
     - Value
   * - Source-to-detector distance
     - 1500 mm
   * - Source-to-isocenter distnace
     - 1000 mm
   * - Detector pixel size
     - 0.338 :math:`\times` 0.338 mm\ :sup:`2`
   * - The number of detector pixels
     - 1024 :math:`\times` 1024
      
We built the 3D Shepp-Logan phantom and contrast phantom to test the CBCT system.
